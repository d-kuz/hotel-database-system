# hotel-database-system
Оперативная база данных и хранилище данных  для гостиницы типа «хостел»

Спроектирована и реализована реляционная база данных для управления гостиничным фондом. Система поддерживает:

- Возможность заселения одного человека, нескольких человек одного пола или семьи (занимает номер целиком)
- Автоматическое определение «пола номера» (мужской/женский) при заселении первого жильца
- Динамическая стоимость проживания в зависимости от категории, вместимости и количества ночей
- Ведение истории клиентов — при повторном заселении данные не заполняются заново
- Учёт дополнительных услуг (кабельное телевидение, доставка еды и др.)

Технологический стек
- СУБД: - Oracle Database
- Язык: SQL, PL/SQL
- Веб-интерфейс: Oracle APEX 

👥 Пользователи системы:
- Администратор - Вывод и изменение списка номеров, их категорий 
- Ресепшионист - Заселение/выселение, управление списком проживающих и клиентов, работа с оплатами и задолженностями
- Менеджер - Управление дополнительными услугами в номерах 
- Клиенты - Просмотр свободных номеров (с разделением по полу) 
- Жильцы - Просмотр доступных дополнительных услуг
